Each guide is built around a job rather than a command. They assume you have run
the [quick start](/getting-started/quick-start/).

- [Add a command](/guides/adding-a-command/) — model a real ali1688 record
  and turn it into a command, a route, and a tool at once.
- [Resource URIs](/guides/resource-uris/) — use ali1688 as a
  `database/sql`-style driver so a host can address ali1688 as
  `ali1688://` URIs.

Add one guide per real task as you build out ali1688.
